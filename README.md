# 📈 Sales Forecasting Dashboard

## Overview

Sales Forecasting Dashboard adalah aplikasi berbasis Streamlit yang digunakan untuk memprediksi jumlah penjualan (Quantity) dan pendapatan (Revenue) pada periode mendatang berdasarkan data historis penjualan.

Proyek ini mengimplementasikan dua algoritma forecasting:

- **Prophet** untuk memprediksi Quantity (Qty)
- **Holt-Winters Exponential Smoothing** untuk memprediksi Revenue

Hasil forecasting disajikan dalam dashboard interaktif yang membantu perusahaan dalam perencanaan stok, pengelolaan inventory, dan penentuan target penjualan.

---
## Installation

Clone repository:

```bash
git clone https://github.com/Asripramudiya/forecesting.git
```

Masuk ke folder project:

```bash
cd forecesting
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Running the Application

Jalankan aplikasi Streamlit:

```bash
streamlit run app.py
```

Aplikasi akan berjalan pada:

```bash
http://localhost:8501
```
